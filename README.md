Инструкция
==========

Зависимости
-----------
1. PHP:
 - & php -r "readfile('https://getcomposer.org/installer');" | php
 - & php composer.phar install
1. Node.JS:
 - & npm install

Использование
-------------
1. Открыть редактор кода:
  - $ subl ./
1. Запустить dev-сервер:
  - $ php -S 127.0.0.1:1337 -t public
1. Перейти в браузер:
  - $ google-crome http://127.0.0.1:1337
